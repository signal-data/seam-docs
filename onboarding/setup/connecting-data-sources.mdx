Connect your data sources to start building your unified account and contact database.

## Where to Connect

In Seam, navigate to **Settings → Connections** to add new integrations.

<Frame>
  <img src="/assets/images/connections-setup.png" alt="Connections setup screen" />
</Frame>

Click **Connect** next to any system to begin the OAuth or API token authorization flow.
